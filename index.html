commission-today.palcu.ro

Commission Today was a project developed during the the EUHackathon in 2014. It's scope was to monitor the recently published meetings of EU Commissioners. You can read more about it's description here. This website was hosting a working version, however the people from Transparency International have taken the idea to a whole new level with the new integritywatch.eu. Our whole source code is open source on Github.

The tool got the attention of several folks on Twitter.
